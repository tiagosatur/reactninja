Intro ao webpack

  • Webpack é um agrupador de módulos (module bundler).
    • O webpack pega os nossos módulos, agrupa em um pacote e entrega um único módulo no formato ES5 para o navegador.
  • Gera o arquivo final que será usado no HTML
  • É utilizado para subir um servidor para desenvolvimento
  • Veremos como utilizar o react hot loader para agilizar o desenvolvimento
